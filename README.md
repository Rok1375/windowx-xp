# WindowX XP

A standalone Windows XP-inspired developer portfolio built with React, TypeScript, and Vite.

## Features
- XP boot and login flow
- Desktop with wallpaper and icons
- Draggable, resizable windows
- Start menu and taskbar
- Window manager with minimize/maximize/close behavior

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

The app builds to `dist/`.
